# Getting Started

This guide will help you get started with VoidUI in your React project.

## Prerequisites

VoidUI is built on top of React and Tailwind CSS. Make sure you have the following installed:

- Node.js 18 or later
- React 18 or later
- Tailwind CSS 3.0 or later

## Installation

1. Install VoidUI using your preferred package manager:

```bash
npm install @voidui/react
# or
yarn add @voidui/react
# or
pnpm add @voidui/react
```

2. Add the VoidUI Tailwind preset to your `tailwind.config.js`:

```js
module.exports = {
  presets: [require('@voidui/react/preset')],
  // ... rest of your config
}
```

3. Import the CSS variables in your global CSS file:

```css
@import '@voidui/react/styles.css';
```

## Basic Usage

Here's a simple example of how to use VoidUI components:

```tsx
import { Button, Card } from '@voidui/react'

export default function App() {
  return (
    <Card>
      <Card.Header>
        <h3>Welcome to VoidUI</h3>
      </Card.Header>
      <Card.Content>
        <p>This is a simple example using VoidUI components.</p>
        <Button variant="primary">Get Started</Button>
      </Card.Content>
    </Card>
  )
}
```

## TypeScript Support

VoidUI is written in TypeScript and includes type definitions out of the box. No additional setup is required.
