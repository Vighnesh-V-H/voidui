# Components

VoidUI provides a comprehensive set of components to help you build beautiful user interfaces. Each component is designed with accessibility and customization in mind.

## Core Components

### Buttons

Buttons are used to trigger actions or navigate between pages.

```tsx
import { Button } from '@voidui/react'

<Button variant="primary">Primary Button</Button>
<Button variant="secondary">Secondary Button</Button>
<Button variant="outline">Outline Button</Button>
<Button variant="ghost">Ghost Button</Button>
```

[Learn more about Buttons →](/docs/components/button)

### Cards

Cards are used to group related content and actions.

```tsx
import { Card } from '@voidui/react'

<Card>
  <Card.Header>
    <Card.Title>Card Title</Card.Title>
    <Card.Description>Optional card description</Card.Description>
  </Card.Header>
  <Card.Content>
    Card content goes here
  </Card.Content>
  <Card.Footer>
    <Button>Action</Button>
  </Card.Footer>
</Card>
```

[Learn more about Cards →](/docs/components/card)

### Input

Input components for collecting user data.

```tsx
import { Input } from '@voidui/react'

<Input placeholder="Enter your name" />
<Input type="email" placeholder="Enter your email" />
<Input type="password" placeholder="Enter your password" />
```

[Learn more about Input →](/docs/components/input)
