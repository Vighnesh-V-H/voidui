# Welcome to VoidUI

VoidUI is a modern, accessible React component library designed to help you build beautiful user interfaces with ease.

## Features

- 🎨 **Beautiful Design**: Modern and clean components that look great out of the box
- ♿ **Accessible**: Built with accessibility in mind following WAI-ARIA guidelines
- 🚀 **Type-Safe**: Written in TypeScript for better developer experience
- 🎯 **Customizable**: Easy to customize with Tailwind CSS
- 📦 **Lightweight**: Minimal bundle size with tree-shaking support

## Quick Start

Install VoidUI using your preferred package manager:

```bash
npm install @voidui/react
# or
yarn add @voidui/react
# or
pnpm add @voidui/react
```

Import and use components in your React application:

```tsx
import { Button } from '@voidui/react'

export default function App() {
  return (
    <Button variant="primary">
      Click me
    </Button>
  )
}
```
